## Tech Stack
This Project was build using Ruby on Rails 7.0.3.

**Client:** HTML5, SCSS, Bootstrap 5, JavaScript ES6

**Server:** Ruby


## Run Locally

Clone the project

```bash
  git clone https://github.com/carlossilveira00/focal_point
```

Go to the project directory

```bash
  cd focal_point
```

Install dependencies

```bash
  yarn install
```

Install Ruby Gems

```bash
  bundle install
```

Start the server

```bash
  rails server
```
